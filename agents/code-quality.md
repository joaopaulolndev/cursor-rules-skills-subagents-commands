# Code Quality Agent

## When to use

- Especialista em qualidade de codigo PHP. Roda PHPStan (Larastan) e Laravel Pint, interpreta erros, corrige automaticamente, gera baselines, e garante que o codigo segue padroes de tipo e estilo.
